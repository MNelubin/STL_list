template<typename NodePtr, typename ValueType, typename DifferenceType, typename Pointer, typename Reference>
class SkipListIteratorImpl< NodePtr, ValueType, DifferenceType, Pointer, Reference >

Implementation of the SkipList iterator. Supports bidirectional iteration.

Template Parameters
NodePtrType of the pointer to the node (e.g., SkipListNode<T>* or const SkipListNode<T>*).
ValueTypeThe type of the value stored in the list.
DifferenceTypeType to represent the difference between two iterators.
PointerPointer to ValueType.
ReferenceReference to ValueType.
